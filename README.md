The Spotify Streaming Insights Dashboard analyzes global music streaming data to uncover trends in streams, artist popularity, track characteristics, and release years.
It helps understand listener behavior and music performance dynamics.

**Objectives:**

Track total streams and music popularity

Analyze danceability and energy trends

Identify top-performing tracks and artists

Compare streaming performance across release years

Understand relationships between track characteristics
